
## About Project

It is a mini Delivery Management System Project. Using Technologies :
- Laravel
- Html
- Css
- Javascript
- Ajax
- Bootstrap

## Flow Of this project

- 1st login to admin
- Then add a product (title, image, price, description).

- Then register a user by submiting the form where Name,email,phone number, address and password only to fill.
- Then It redirect to the user login page then user login using emailid and password .then enter in dashboard.
- In dashboard their is a product listing page , go to that page then u see the products.
- In that product card see a place order btn when you click that it opens a modal.
- In that modal per unit price showing , then you put how many units you want to order then after fill units under that show the total price .
- After that we have a payment option you can place order using online payment or cash on delivery.
- If you choose cash on delivery it directly place the order , if you choose online it open payment getway.
- After place a order it check stock for that product if required quantity is less than stock then no issue order is placed but if required quantity is greterthan the stock it place order but status is hold .
- After admin change the stock it will automatically resume.
- After order place admin need to give permision accept or reject the order.
- If accept the order then we are eble to deliver that order
- After approval admin manually can change the status for order like Pending,Processing,Dispatched,Delivered.
